# AmeriContacts 

## Summary

This is a simple web application to manage a list of contacts, created at the
request of AmeriSave Mortgage Corporation.

## Technologies Used

* Spring Boot
* Spring MVC
* Maven
* Thymeleaf
* Bootstrap
* HTML
* CSS
